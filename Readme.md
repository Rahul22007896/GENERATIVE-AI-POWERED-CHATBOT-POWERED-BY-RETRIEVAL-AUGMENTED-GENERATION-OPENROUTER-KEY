#  Generative AI Chatbot (RAG + OpenRouter)

An intelligent chatbot powered by **Generative AI** and **Retrieval-Augmented Generation (RAG)** that delivers accurate, context-aware responses by combining Large Language Models with external knowledge sources.

---

## Features

*  **Retrieval-Augmented Generation (RAG)** for improved accuracy
*  **PDF-based knowledge ingestion** (custom data support)
*  **Context-aware responses** using LLMs
*  **OpenRouter API integration** for flexible model access
*  **Simple UI interface** for user interaction
*  Lightweight and easy to deploy

---

---

## Tech Stack

* **Python**
* **RAG Pipeline**
* **OpenRouter API (LLMs)**
* **HTML / CSS / JS (Frontend UI AND LOGIC)**
* **PDF Processing (Knowledge Base)**

---

##  How It Works

1.  User uploads or uses a predefined document (PDF)
2.  Text is split into chunks
3.  Relevant chunks are retrieved based on query
4.  LLM generates response using retrieved context
5.  Response is displayed in UI
